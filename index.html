In this project, we utilized Python Flask to create an application centered around showing baseball statistics. In this project, I was mainly focused on developing the backend of the project, such as the login and database system, page management, and the APIs. Location Protocol Server/Client System

In this project, I developed a server and client that users can use to input their location and have that location populate on a map. The server/client communicate with each other using TCP and UDP protocols depending on certain factors. TCP messages are used when users create/change their locations, while UDP messages are used to update the location of all other users when one changes their location. This project was created for the Data Communications course.
